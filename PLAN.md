## הפערים והפתרונות

### 1. קול חוזר (TTS) — הפער הכי גדול בעברית
- **edge-tts** (פייתון, חינם): קולות מיקרוסופט מעולים בעברית — אבל ענן (לא פרטי).
- **Piper** (מקומי, מהיר): איכות סבירה, תמיכה עברית חלשה.
- **MMS-TTS-heb / Roboshaul**: עברית מקומית, איכות בינונית.
- 🎯 המלצה: להתחיל עם edge-tts (חוויה טובה מיד), להחליף למקומי כשהתחום יבשיל.
  זו פשרת הפרטיות היחידה ששווה לשקול בהתחלה.

### 2. מוח סוכן עם כלים (Tool Calling)
- gemma3 חלש ב-tool calls; **qwen2.5:14b** או **llama3.1:8b** תומכים function-calling דרך Ollama.
- לולאת סוכן פשוטה משלנו (בלי LangChain): שאלה → המודל בוחר כלי → מריצים → התשובה חוזרת למודל → תשובה קולית.
- 🎯 טיפ קריטי: לדבר עברית עם המשתמש, אבל שמות כלים/JSON באנגלית — מודלים קטנים מתבלבלים בעברית טכנית.
- אופציה: מוח ענן (Claude API) לשאלות מורכבות + מקומי לפשוטות — hybrid. לשיקולך.

### 3. כלים — מה Jarvis יידע לעשות (לפי סדר מימוש)
1. **מחשב**: שעה/סוללה/נפח דיסק, פתיחת אפליקציות, חיפוש קבצים (Everything CLI — מיידי), צילום מסך.
2. **רשת**: מזג אוויר, חיפוש (Brave Search API חינמי / SearxNG מקומי), קריאת דפים.
3. **Google** (OAuth אישי, חינם): Gmail — "מה המיילים החדשים?", סיכום; Calendar — "מה יש לי מחר?"; Drive.
4. **זיכרון אישי**: SQLite + embeddings (nomic-embed דרך Ollama) — "תזכור ש...", "מה אמרתי על...".
5. **ראייה**: qwen2.5-vl דרך Ollama — "מה על המסך שלי?" (תיאור, לא שליטה).

### 4. תצוגת נתונים ("ותראה לי")
- להרחיב את הווידג'ט לחלון תשובות: כרטיסי מידע (מייל, פגישה, מזג אוויר) + טקסט.
- 🎯 בהמשך: dashboard מקומי בדפדפן (localhost) לתשובות עשירות — טבלאות, גרפים.

### 5. שיחה טבעית
- שלב 1: push-to-talk (יש לנו).
- שלב 2: **wake word** — openWakeWord עם מודל "hey jarvis" מוכן! (באמת קיים)
- שלב 3: שיחה רציפה עם VAD — הוא מקשיב אחרי שהוא עונה.

## מה לא ריאלי (שתדע מראש)
- ❌ **קטיעה באמצע (barge-in)** אמינה — קשה מאוד בלי echo cancellation רציני.
- ❌ **שליטה חופשית ב-GUI** ("תזמין לי טיסה") — computer-use מקומי עם מודל 14B יהיה איטי ושבור. תיאור מסך כן, שליטה לא.
- ❌ **משימות ארוכות אוטונומיות** — מודל 14B מתפזר אחרי 3-4 צעדים. Jarvis שלנו = פקודות קצרות וחדות.
- ⚠️ עברית ב-tool-calling פחות אמינה מאנגלית — נפצה בפרומפטים ודוגמאות.
